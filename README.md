# Crawling_By_Python

## 다방방정보크롤링.py
실행을 한다
(코드는 셀단위 실행으로 작성 되어있는데 셀 구분을 모두 지우고 한 번에 실행해도 무관하다.)

다방방정보크롤링은 다방에 정보를 얻고 싶은 지역을 터미널에 입력을 한다.

전세값, 월세값, 방크기, 한달생활비(이는 다방에서 제공하는 정보로 관리비 인듯 하다.),집의주소, 다방웹주소를 저장해준다.

월세, 전세를 따로 구분하여 json파일로 저장한다.
